<time datetime="2025-12-09">Martedì 9 dicembre 2025</time>, ho tenuto una presentazione durante il 58° episodio di [Bologna Front-end](https://www.meetup.com/bologna-front-end/) presso Granata a Bologna.

Il talk, dal titolo "Ehi Gemini, sistema il CSS!", è stato una demo live del [Chrome DevTools MCP Server](https://github.com/ChromeDevTools/chrome-devtools-mcp) — uno strumento che permette di controllare Chrome DevTools attraverso il [Model Context Protocol](https://modelcontextprotocol.io/) (MCP), utilizzando comandi in linguaggio naturale direttamente dal terminale.

Ho mostrato come sfruttarlo per:

- Eseguire audit di accessibilità
- Effettuare il debug di layout CSS complessi
- Simulare diverse condizioni di rete
- Testare viewport personalizzati per il responsive design

Tutto attraverso il linguaggio naturale, senza mai lasciare il terminale. Davvero notevole!
