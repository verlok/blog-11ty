<time datetime="2026-02-25">Mercoledì 25 febbraio 2026</time>, ho tenuto una presentazione al [Bologna JS Meetup](https://www.meetup.com/bologna-js-meetup/) presso Spazio01 Coworking a Bologna.

Il talk, dal titolo "Gemini + Chrome: Oltre la Chat. Usare l'AI per controllare il browser", è stato una demo del [Chrome DevTools MCP Server](https://github.com/ChromeDevTools/chrome-devtools-mcp) — mostrando come connettere l'IA a Chrome DevTools attraverso il [Model Context Protocol](https://modelcontextprotocol.io/).

Ho mostrato esempi pratici di come:

- Leggere il DOM in tempo reale e manipolare le pagine web
- Eseguire smoke test visivi su diversi viewport
- Simulare interazioni utente complesse per test end-to-end
- Intercettare richieste di rete e analizzare metriche di performance
- Identificare colli di bottiglia nelle prestazioni
