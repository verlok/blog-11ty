Let me take a step back. Antigravity 1 was essentially a VS Code shell with a built-in chat. It felt quite similar to the VS Code + Copilot setup, but with a highly interesting twist: **Planning Mode**. Before executing any changes, the chat would pause, reflect back what it understood, show you a detailed step-by-step implementation plan, and ask for explicit confirmation. It was very similar to Claude's planning mode but packaged in a much cleaner, easier-to-read, and commentable user interface.

With **Antigravity 2**, they’ve taken a bold step. They’ve completely removed the code editor mode (or they’ve hidden it so well that I haven't found it yet!). Instead, it is a pure, distraction-free chat interface focused entirely on the agentic workflow, while keeping that wonderfully convenient Planning Mode from version 1 completely intact.

## Seamless Multi-Project Workflows

What makes Antigravity 2 so powerful is how it operates in the background:

*   **Multi-Project Support**: It runs seamlessly across your active workspace projects.
*   **Parallel Chats**: Each project can have multiple concurrent chats running in parallel. 
*   **Git Worktrees**: Each independent chat can run directly on your local branch or inside its own dedicated Git worktree, allowing the agent to write code and run builds in isolation without locking up your main working tree.

I recommend using it side-by-side with your favorite traditional IDE. In my day-to-day work, I constantly jump between Antigravity 2 and **JetBrains PyCharm**. I use Antigravity to run agent tasks, plan refactoring, and modify code, and then I switch over to PyCharm to run, test, and handle everything else just like I did before. It’s the perfect hybrid workspace.

## Under the Hood: Gemini 3.5 Flash is Fire

Antigravity 2 supports both Gemini 3.5 and Claude. If you have a Google account, you get a generous free tier of Gemini 3.5 included right out of the box, which I highly recommend trying. If you prefer Claude, you can simply connect your Anthropic account and go.

I’ve personally tested almost every major model out there—from OpenAI's o3, to Gemini 2.5 Pro, to Claude Sonnet 3.7 and Opus 4.6. Over time, I’ve consistently found myself appreciating the quality and relevance of Gemini's answers the most. 

And **Gemini 3.5 Flash**? It is absolute fire (or as we say in Italy, *una bomba*). It is blisteringly fast, incredibly accurate, and highly intelligent. You really have to try it yourself to understand the speed difference.
